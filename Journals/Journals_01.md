Today is 12th July, 2026. Time is 8:21PM. Currently; starting working on a project that I wanted to work on for a while. its like a dream project for me. I'm calling "Saim-Mias", Its a Custom Made Mouse for myself... I'll explain it in detail later on.

The is the start of the journal, I'll be logging everything here in Obsidian's Vault that will live inside Cryptomator's Encrypted Vault to sync all the work on my other devices.|

Now Lets Start the actual work here.
First lets answer the main question her; Why the Hell I'm making a mouse in the first place?
Answer is very simple and clear: I have a Gaming Mouse which I bought after I saw it in my friend's place, I really loved it... Then I asked him where did you get it from? I went to that place and Found it. After one year of using it, I encountered an issue, The mouse was RGB and it had no Software nor any option on the hardware side to change the RGB neon lightning effect. ANd that always pisses me off. I really love this mouse build and quality and core performance. But No control on the RGB was pain in the rear.

So here I am Started working on the mouse project. This idea occurred to be a while back when I was working on my first hardware project funded by Hack Club Blueprint. The project was a grate success.  You can take a look at it: [ModuPad](https://github.com/lassiipk/ModuPad).

Anyways lets continue discussing the project,  After some research on my mouse i found out that my mouse is not capable of doing such action, also know as ARGB, It stands for Addressable Red Green Blue lightening effect. the core problem was with the microcontroller and non-ARGB lights. This was all figured and concluded after opening the mouse to view its internal components, I Also have some images to display the problem, I'll make sure to show them for you to look into it if you are interested init.
